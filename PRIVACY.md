# Política de Privacidad

**Última actualización:** 02/01/2026

Esta Política de Privacidad describe cómo la aplicación **LibreAuth** ("nosotros", "nuestro" o "la Aplicación") maneja tus datos.

Creemos firmemente en la privacidad del usuario. Nuestra política es simple: **No recopilamos, almacenamos ni compartimos tus datos personales.**

## 1. Recopilación y Uso de Datos

### Datos Personales
La Aplicación funciona de manera **100% offline (sin conexión)**. No recopilamos, transmitimos ni almacenamos en servidores externos ninguna información personal, claves, contraseñas o datos de uso.

### Datos de las Cuentas (Tokens 2FA)
Toda la información que introduces en la Aplicación (nombres de cuentas, claves secretas, configuraciones) se almacena **exclusivamente en tu dispositivo** utilizando el almacenamiento seguro del sistema (`SecureStore` en iOS y Android). Estos datos están encriptados y nunca salen de tu teléfono.

Si eliminas la Aplicación o borras los datos de la misma, esta información se perderá permanentemente y no podrá ser recuperada por nosotros, ya que no tenemos copias.

## 2. Permisos del Dispositivo

La Aplicación puede solicitar los siguientes permisos para funcionar correctamente:

* **Cámara:** Este permiso se utiliza **únicamente** para escanear códigos QR con el fin de añadir nuevas cuentas de autenticación. Las imágenes capturadas por la cámara se procesan en tiempo real en el dispositivo para extraer la información del código y **nunca se guardan ni se envían a terceros**.

## 3. Servicios de Terceros

La Aplicación ha sido construida utilizando **Expo** y **React Native**. Aunque la Aplicación en sí no recopila datos, el sistema operativo (Android/iOS) o las librerías estándar podrían recopilar estadísticas técnicas anónimas (como informes de fallos o "crashes") si el usuario ha dado su consentimiento al sistema operativo para compartir diagnósticos.

## 4. Cambios en esta Política de Privacidad

Podemos actualizar nuestra Política de Privacidad de vez en cuando. Te recomendamos revisar esta página periódicamente para ver si hay cambios. Notificaremos cualquier cambio publicando la nueva Política de Privacidad en esta página.
